[name]

Represents a lookup table for colormaps. It is used to determine the color values from a range of data values.

Code Example

// predefined colormap 'rainbow' lookup var lut = new Lut( 'rainbow', 512 ); var color = lut.getColor( 0.5 ); // custom colormap lookup var lut = new Lut( [ [ 0.0, 0x1019f7 ], [ 0.2, 0x0011e1 ], [ 0.4, 0x115cca ], [ 0.6, 0x33a5c5 ], [ 0.8, 0x25cea5 ], [ 1.0, 0x16f782 ] ], 512 ); var color = lut.getColor( 0.5 );

Constructor

[name]( [param:String map], [param:Integer numberOfColors] )

[name]( [param:Array map], [param:Integer numberOfColors] )

map - The name of predefined colormap or Array of color stops. Optional; Default is "rainbow". Other predefined colormaps: "cooltowarm", "blackbody" and "grayscale".
numberOfColors - Number of colors used to represent the data array. Optional; Default is 32.

Properties

[property:Array colors]

The palette. Array of [page:Color]s

[property:Array map]

The colormap. Array of color stops. Used to construct the palette.

// Color stop examples [ 0.0, 0xff0000 ] [ 0.0, 'red' ] [ 0.0, new Color('red') ]

[property:Float minV]

The minimum value to be represented with the lookup table. Default is 0.

[property:Float maxV]

The maximum value to be represented with the lookup table. Default is 1.
